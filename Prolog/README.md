<h1>Aplicação em PROLOG</h1>

<h2>Para rodar o projeto é necessário que seja executado no terminal os seguintes comandos:</h2>
<ol>
    <li>```cd Desktop
    <li>git clone https://github.com/amaurineo/PLPProjetoOverBooking.git
    <li>cd Prolog
    <li>swipl -f .\main.pl```
</ol>
